# Innovative Website Project

## Overview
This project is an innovative website designed to showcase modern web development techniques using only HTML and CSS. The website features a responsive layout, a navigation bar with a dropdown menu, and a footer with social media links.

## Project Structure
```
innovative-website
├── src
│   ├── index.html
│   ├── styles
│   │   └── main.css
│   ├── components
│   │   ├── navbar.html
│   │   └── footer.html
│   └── assets
│       └── fonts
│           └── custom-font.css
├── README.md
```

## Features
- **Responsive Design**: The website is fully responsive, ensuring a seamless experience on devices of all sizes.
- **Custom Fonts**: Utilizes custom fonts for enhanced typography.
- **Dropdown Navigation**: The navigation bar includes a dropdown menu for easy access to various sections.
- **Social Media Links**: The footer contains links to social media platforms for user engagement.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `src/index.html` file in your web browser to view the website.
3. Modify the CSS in `src/styles/main.css` to customize the appearance as needed.
4. Update the navbar and footer components in `src/components/navbar.html` and `src/components/footer.html` respectively.
